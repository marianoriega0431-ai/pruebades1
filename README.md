#  Student Management System (CRUD with CSV)

##  Description
This project is a **Student Management System** developed in Python that allows users to manage student records using basic CRUD operations:

- Create  
- Read  
- Update  
- Delete  

All data is stored in a CSV file, ensuring persistence even after the program is closed.

---

##  Features

-  Create new students  
-  View all students  
-  Search students by ID  
-  Update student information  
-  Delete students  
-  Automatic saving in a CSV file  

---

##  Technologies Used

- Python 3  
- CSV module  
- OS module
